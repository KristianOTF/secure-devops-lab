# Secure DevOps Lab Architecture

## Overview

The goal of this project is to simulate a small production infrastructure using modern DevOps and DevSecOps practices.

The lab will progressively evolve from a single Linux server into a complete cloud-native environment.

---

## Components

- Linux Server
- Docker Engine
- Reverse Proxy (Nginx)
- Sample Web Application
- PostgreSQL Database
- Monitoring Stack
- CI/CD Pipeline
- Infrastructure as Code (Terraform)

---

## Future Components

- Kubernetes Cluster
- AWS Deployment
- Secrets Management
- Vulnerability Scanning
